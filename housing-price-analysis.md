# 房价趋势分析 Skill

## Overview

本 skill 通过 **5 维度分析框架**，系统评估某个城市的房价走势，判断是短期波动还是长期趋势，并为普通人提供可操作的建议。

所有输出使用**大白话**，避免专业术语堆砌，确保普通人能看懂。

## When to Use This Skill

**在以下场景自动加载此 skill：**

- 用户询问某个城市的房价走势（如"XX房价未来会涨还是跌"）
- 用户问是否应该在某个城市买房/卖房
- 用户想了解某个区域的房地产市场前景
- 用户提到"房价趋势"、"楼市分析"、"买房建议"等关键词

---

## 分析框架：5 维度模型

### 维度总览

| # | 维度 | 核心问题 | 权重 |
|---|------|---------|------|
| 1 | 人口 | 这个城市的人在增加还是减少？ | 30% |
| 2 | 土地供应 | 房子够不够卖？ | 20% |
| 3 | 政策 | 有没有政策刺激/打压？ | 15% |
| 4 | 经济与收入 | 老百姓兜里有没有钱？ | 25% |
| 5 | 租售比 | 房价贵不贵？ | 10% |

## 执行流程

### Step 1: 确定分析对象

从用户提问中提取：
- **目标城市**（如"武汉"、"成都"）
- **具体区域**（如有，如"光谷"、"天府新区"）
- **时间关注点**（短期/中期/长期）
- **用户身份**（自住/投资/观望）

### Step 2: 数据收集（WebSearch）

使用 WebSearch 收集数据，**合并搜索，控制在 3 次以内**：

| 搜索批次 | 搜索关键词模板 | 目标数据 |
|---------|---------------|---------|
| 第1次 | "{城市名} 2024 2025 常住人口 净流入 GDP 经济增速" | 人口 + 经济 |
| 第2次 | "{城市名} 2024 2025 房价走势 新房 二手房 成交量 租售比 土地供应" | 房价 + 土地 |
| 第3次（可选） | "{城市名} 2024 2025 房地产政策 限购 取消" | 政策 |

**必须收集的数据点：**

#### 维度1 - 人口
- 常住人口总量及同比变化（近2-3年）
- 人口净流入/净流出
- 在校大学生数量（如有）
- 城镇化率

#### 维度2 - 土地供应
- 新房成交量及同比变化
- 二手房成交量及同比变化
- 库存/去化周期（如有）

#### 维度3 - 政策
- 限购政策状态
- 首付比例
- 房贷利率变化
- 重大楼市新政

#### 维度4 - 经济
- GDP 总量及增速
- 产业结构（三次产业占比）
- 高技术制造业/新兴产业增速
- 与全国平均增速对比

#### 维度5 - 租售比
- 新房价格环比/同比变化
- 二手房价格环比/同比变化
- 典型区域租售比

### Step 3: 逐维度分析

#### 维度1 - 人口判断标准

| 信号 | 判断 |
|------|------|
| 常住人口连续3年净流入，年均增量 > 5万 | 强支撑（长期看涨） |
| 常住人口连续3年净流入，年均增量 1-5万 | 稳健支撑 |
| 人口数据忽上忽下，无明确趋势 | 中性 |
| 常住人口净流出 | 长期看跌 |

#### 维度2 - 土地供应判断标准

| 信号 | 判断 |
|------|------|
| 核心区土地几乎无新增供应 | 供给紧张（支撑房价） |
| 新房成交量 > 新房供应量 | 去库存良好 |
| 新房成交量 < 新房供应量 | 库存压力 |
| 大量新楼盘集中入市 | 短期供给过剩 |

#### 维度3 - 政策判断标准

| 信号 | 判断 |
|------|------|
| 限购放松/取消、首付降低、利率下降 | 短期刺激（不可持续） |
| 限购收紧、首付提高、利率上升 | 短期打压 |
| 房产税出台 | 长期利空 |
| 重大城市规划落地（地铁、产业） | 长期利好 |

#### 维度4 - 经济判断标准

| 信号 | 判断 |
|------|------|
| GDP增速持续高于全国 | 经济强（支撑购买力） |
| 产业结构升级（高技术制造业增速 > 10%） | 新动能强 |
| GDP增速低于全国或持续放缓 | 经济弱 |
| 产业衰退（资源枯竭、工厂外迁） | 长期看跌 |

#### 维度5 - 租售比判断标准

| 租售比 | 判断 |
|--------|------|
| < 1.5% | 房价偏高，可能存在泡沫 |
| 1.5% - 3% | 相对合理 |
| > 3% | 可能被低估 |

### Step 4: 综合评分与结论

#### 评分汇总表

| 维度 | 评分(1-5星) | 权重 | 方向 |
|------|------------|------|------|
| 人口 | | 30% | 支持/拖累/中性 |
| 土地供应 | | 20% | 支持/拖累/中性 |
| 政策 | | 15% | 支持/拖累/中性 |
| 经济 | | 25% | 支持/拖累/中性 |
| 租售比 | | 10% | 支持/拖累/中性 |

#### 结论模板

| 时间维度 | 判断 | 理由 |
|---------|------|------|
| 短期（1年内） | 筑底/上涨/下跌/波动 | 基于政策+近期成交量 |
| 中期（1-3年） | 分化/企稳/上涨/下跌 | 基于人口+经济+库存 |
| 长期（3-5年+） | 温和上涨/横盘/下跌 | 基于全部维度综合判断 |

#### 区域分化分析（如数据允许）

| 区域 | 判断 | 理由 |
|------|------|------|
| 核心区A | 看涨/中性/看跌 | 产业/配套/供给分析 |
| 远郊区B | 看涨/中性/看跌 | 产业/配套/供给分析 |

### Step 5: 给普通人的建议

根据分析结果，针对不同身份给出建议：

| 身份 | 建议方向 |
|------|---------|
| 刚需自住 | 是否可以入手？注意什么？ |
| 改善置换 | 是否是好的置换窗口？怎么换？ |
| 投资 | 是否建议投资？如果投资看哪里？ |
| 观望 | 建议等多久？关注什么信号？ |

## 输出格式要求

### 整体结构

```
# {城市名}房价未来走势分析——用5个维度判断

## 一、逐项分析
### 1. 人口维度
### 2. 土地供应维度
### 3. 政策维度
### 4. 经济与收入维度
### 5. 租售比维度

## 二、综合评分

## 三、结论（短/中/长期）

## 四、区域分化分析（如数据允许）

## 五、给普通人的建议

## 数据来源
```

### 语言风格

- **大白话为主**：避免堆砌专业术语
- **多用比喻**：帮助理解
- **表格优先**：关键信息用表格呈现
- **有结论有建议**：不只是分析，要给出可操作的行动指南
- **标注数据来源**：每个数据点注明出处

### 核心原则

1. **数据说话**：所有判断必须有数据支撑，不凭感觉
2. **区分短长期**：明确告诉用户什么是短期波动、什么是长期趋势
3. **承认不确定性**：不说"一定会涨/跌"，而是说"大概率"、"倾向于"
4. **区域分化**：不做一刀切判断，同一城市不同区域可能完全不同
5. **实用导向**：最终要回答"那我该怎么办"

## 数据来源参考

优先搜索以下官方数据源：

- **人口数据**：各市统计局《国民经济和社会发展统计公报》
- **经济数据**：各市政府官网、统计局
- **房价数据**：国家统计局70城房价指数
- **政策信息**：各市住建局、政府官网
- **土地/成交数据**：各市住房和城市更新局、中国指数研究院
